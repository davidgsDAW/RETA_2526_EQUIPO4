## 📝 Descripción del Proyecto

El nombre del reto que se nos ha propuesto es __'Gestión y Localización del Material del Taller de Informática'__, y consiste en el desarrollo de una aplicación de escritorio que gestione el inventario del taller de informática de nuestro centro, IES Miguel Herrero Pereda.

Junto a esta aplicación, nuestro objetivo es desarrollar una web y una base de datos alojados en servidores dentro de máquinas virtuales.
